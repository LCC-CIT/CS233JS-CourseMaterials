-  **Statewide team for Supercomputing 26 competition**
   
   A collaboration between the higher education institutions in Oregon is forming a team to compete in **[SCC Connect](https://sc26.supercomputing.org/students/student-cluster-competition/)** at [**SC26**.](https://sc26.supercomputing.org/) In the SCC Connect, student teams design and build small compute clusters, learn to compile and use scientific applications, and apply optimization techniques for their chosen architectures in a non-stop, 48-hour challenge using cloud resources. This is a hands-on, real-world experience mentored by high performance computing (HPC) professionals from the state of Oregon. 
   
   If interested, please reach out to **[oit-oregonscc-group@pdx.ed](mailto:oit-oregonscc-group@pdx.edu)**. There are no prerequisites to participate other than a willingness to learn. **Students from both STEM and non-STEM backgrounds are welcome** Even if you do not wish to participate in the competition, the training and mentorship is available and  free. 

### This Week's Learning Objectives

- To be able to use of *git* and GitHub for software version control.
- To introduce you to object oriented programming (OOP) with *classes* (an ES6 feature).
  - How to do OOP program design.
  - How to represent OOP designs in a Unified Modeling Language (UML) class diagram.
