# CS233JS-CourseMaterials

In CS 233JS, Intermediate Programming: JavaScript, students learn features of JavaScript introduced in ES6, includeing ES6 classes, as well as how to use Node.js, <s>Webpack, Babel</s> Vite, Vitest and ohter JS dev tools. The prerequisite for this course is CS 133JS, Beginning Programming: JavaScript, here is a link to the [CS 133JS course materials repository](https://github.com/LCC-CIT/CS133JS-CourseMaterials). 

You may access documents in this repository via [the repository's web address](https://lcc-cit.github.io/CS233JS-CourseMaterials).

**Spring 2026: I am currently in the process of revising and updating this course.**  

- The lab solution and starter file repositories listed below are in the process of being revised or replaced. 
- I am no loger using GitHub Classroom.
