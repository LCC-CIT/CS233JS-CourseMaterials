## CS233JS-LN-W01-D0-Overview.md

- To introduce the use of Bootstrap for styling web pages.

## CS233JS-LN-W02-D0-Overview-w26.md

## CS233JS-LN-W02-D0-OverviewAndAnnouncements.md
- To introduce you to key features of a "new" version of JavaScript - ES6, including classes.
- To introduce the use of git and GitHub for software version control
- To develop your proficiency in designing, implementing, testing and debugging client-side web-based applications written in JavaScript

## CS233JS-LN-W03-D0-Overview.md
- to introduce you to additional features of JavaScript that were added in the ES6 (2015) release.
  - Arrow functions
  - Default parameters
  - Template literals
  - Array constructor
  - Destructuring assignment
- to introduce the use of *git* and GitHub for software version control.
- to introduce you to object oriented programming (OOP) with *classes* (an ES6 feature).
  - How to do OOP program design
  - How to represent OOP designs in a Unified Modeling Language (UML) class diagram.

## CS233JS-LN-W04-D0-Overview.md
- To introduce you to commonly used JavaScript development tools including Node.js and Webpack.
- To introduce you to using the web browser's LocalStorage for storing client-side data.
- To develop your proficiency in designing, implementing, testing and debugging JavaScript web apps.

## CS233JS-LN-W05-D0-Overview.md
