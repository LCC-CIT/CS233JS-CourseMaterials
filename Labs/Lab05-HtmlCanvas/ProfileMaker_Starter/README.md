# Profile Maker Starter

This is a starter project for the Profile Maker lab assignment in CS233JS. 

## Included Features

This project provides the basic infrastructure to get you started:
- A Controller-Service architectural pattern (`profileUiController.js` and `profileLogicService.js`).
- HTML5 Canvas rendering setup.
- Image file upload processing.
- A functional "Download" button to save the canvas as a PNG.
- Automatic image scaling to fit the canvas proportionally.

## Your Task

Your objective is to expand this starter project into a fully-featured Profile Maker by adding advanced features such as:
- A circular clipping mask.
- Custom drawing tools (freestyle drawing).
- Emoji stickers that can be dragged and placed on the canvas.
- Decorative borders and styles.
- Image filters or visual effects.
- Curved badge text.

## Development Environment

- **Build System**: [Vite](https://vitejs.dev/)
- **Logic Standards**: ES6+ JavaScript modules.

### Getting Started

1. Run `npm install` to install dependencies.
2. Run `npm run dev` to start the local development server.
